OPC 开放接入

完整文档见仓库 docs/AI_OPC_INTEGRATION.md。

接入流程

  1. 向 opl 平台申请 appId / appSecret
  2. 配置 opc_connect_app（API 或后台）
  3. 用户访问 /open/oauth2/opc/authorize?appId=...
  4. 回调 /open/oauth2/callback 完成本地绑定与登录

首次登录

跨平台首次授权且未登录时，回调后进入绑定选择页：用户可选择「登录已有账号并绑定」或「创建新账号并绑定」（与经典 OAuth 一致）。

OPC 在 callback 校验 Session 绑定的 state；Open API 须 Bearer 令牌。
