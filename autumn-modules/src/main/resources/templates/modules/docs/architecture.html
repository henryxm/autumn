架构与模块

本页用于让开发者先理解系统边界，再进入各模块细节。

工程分层

  • autumn-lib：公共能力与基础工具
  • autumn-modules：业务核心模块（sys/gen/job/oauth/wall 等）
  • autumn-starter：一键引用框架功能
  • web：系统启动入口与页面承载

典型请求链路

  • 前端页面（templates/modules/...）
  • Controller 接口路由
  • Service 业务处理
  • Dao 数据访问

权限与数据权限

  • 菜单权限控制到页面与按钮级
  • 数据权限按角色、部门和本人数据范围控制
  • 管理员具备系统级操作能力

文档补齐建议

  • 补充系统上下文图（模块依赖关系）
  • 补充权限矩阵（角色 x 菜单 x 接口）
  • 补充配置项索引（按模块归类）
