OPL 开放平台对接

完整文档见仓库 docs/AI_OPL_INTEGRATION.md。

端点速查

  • 开发者 API：POST /open/api/v1/platform/account/open、/app/register
  • OAuth2：/open/oauth2/authorize → /token → /userInfo（可选 PKCE；确认页 CSRF）

标识说明

  • appId：应用唯一标识（独立于 OAuth client_id）
  • openId：opl_open_identity，App（appId）内唯一
  • unionId：opl_open_union，开发者账号下唯一
