代码生成（gen）

用于通过数据表快速生成基础代码结构，降低 CRUD 开发成本。

功能入口

  • 生成器：/gen/generator/list、/gen/generator/code、/gen/generator/reset
  • 生成类型：/gen/gentype/list、/gen/gentype/save、/gen/gentype/copy
  • 模板文件（resources/template/*.vm）

推荐使用流程

  • 先定义数据表和命名规范
  • 配置模块名、包名、表前缀
  • 执行生成并检查输出代码
  • 在生成代码上补业务逻辑，不直接改模板默认输出结构
建议补充“表设计规范”和“模板定制规范”，避免不同开发者生成风格不一致。

关键配置/参数

参数/配置 示例 说明
moduleName order 生成目标模块名
tablePrefix tb_ 表名前缀过滤
moduleText 系统管理模块 生成类型中展示名称，用于区分不同业务模板

API 示例（真实路径）

GET /gen/generator/list?page=1&limit=10&tableName=sys_user
POST /gen/generator/code {"tables":["sys_user"],"moduleName":"sys"}
POST /gen/gentype/copy {"id":1,"moduleName":"demo"}
{"code":0,"msg":"success","page":{"list":[...],"currPage":1}}

页面入口（templates）

  • /modules/gen/generator -> templates/modules/gen/generator.html
  • /modules/gen/gentype -> templates/modules/gen/gentype.html

权限码清单

gen:generator:list, gen:generator:code, gen:generator:reset
gen:gentype:list, gen:gentype:info, gen:gentype:save, gen:gentype:update, gen:gentype:delete, gen:gentype:copy

最小可跑通用例

  1. 确认当前账号具备 gen:generator:list 与 gen:generator:code。
  2. 访问 /modules/gen/generator，输入表名执行查询。
  3. 调用 GET /gen/generator/list?page=1&limit=10&tableName=sys_user，确认返回表信息。
  4. 选择一张表执行代码生成，确认下载或返回内容成功。

故障排查

现象可能原因处理建议
生成失败表不存在或无访问权限先在数据库校验表结构与连接账号权限
生成代码不可编译模板与当前实体字段不匹配检查 vm 模板变量与字段命名
