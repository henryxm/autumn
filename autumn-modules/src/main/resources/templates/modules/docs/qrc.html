扫码登录（QRC）

统一扫码登录/授权编排模块，支持同源 PC 登录与 OAuth 第三方授权。实现包：cn.org.autumn.modules.qrc。

完整业务流程（数据关系、拓扑图、四种场景分步说明）请阅读： 授权与扫码登录流程指南

客户端对接（仓库 docs/）

文档读者
AI_QRC_CLIENT_API.mdAPP：扫码、确认页、伪代码、联调清单
AI_QRC_API.md全部 HTTP 端点、报文、状态机、错误码
AI_QRC_INTEGRATION.md第三方：OAuth Redirect / 自建 QR / delivery
AI_OAUTH_INTEGRATION.mdOAuth2 token、userInfo、账号绑定
AI_QRC.md模块结构、配置、安全基线

HTTP 路径与报文

前缀Controller报文鉴权

/qrc/scanticket/*ScanTicketControllerGenR后台权限

Intent（业务场景）

Intent说明

OAUTH_CONSENT已登录 + 手机二次确认
OAUTH_DEVICE第三方 Open API 自建 QR
