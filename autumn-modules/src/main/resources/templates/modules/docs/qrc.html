扫码登录（QRC）

统一扫码登录/授权编排模块，支持同源 PC 登录与 OAuth 第三方授权。实现包：cn.org.autumn.modules.qrc。

完整业务流程（数据关系、拓扑图、四种场景分步说明）请阅读： 授权与扫码登录流程指南

文档（仓库 docs/）

  • AI_QRC.md — 对接指南、代码结构、配置与安全
  • AI_QRC_API.md — Web / APP / Open / QR 链接 / OAuth 端点
  • AI_OAUTH_INTEGRATION.md — OAuth2 第三方对接（授权码、token、userInfo、账号绑定）
  • AI_QRC_INTEGRATION.md — 第三方集成模式（含扫码分支）
  • AI_QRC_CLIENT_API.md — APP 扫码确认手册

HTTP 路径

前缀Controller鉴权
/qrc/web/v1/*ScanWebController匿名
/qrc/api/v1/*ScanAppApiController@Authenticated
/qrc/open/v1/*ScanOpenApiControllerclient 密钥
/qrc/v1/t/{uuid}ScanLinkController匿名（仅公开字段）
/oauth2/authorizeAuthorizationController统一 login.html

Intent（业务场景）

Intent说明
SELF_WEB_LOGIN网站扫码登录（PC Session）
OAUTH_AUTHORIZE未登录 OAuth 授权扫码
OAUTH_CONSENT已登录 + 手机二次确认
OAUTH_DEVICE第三方 Open API 自建 QR
