多语言（lan）

用于管理系统支持语言、语言资源与界面切换。

功能入口

  • 语言管理：/lan/language/list|save|update|delete
  • 支持语言：/lan/language/supportedlist、/lan/language/updatesupportedlanguage
  • 语言写入：POST /lan/language/put
  • 开放接口：/api/getsupportedlanguage

规范建议

  • 统一翻译键命名风格（模块_页面_含义）
  • 新增语言时先补核心页面与错误提示
  • 建立缺失翻译扫描机制

关键配置项（来自 sys_config.param_key）

配置键 默认值 说明
UPDATE_LANGUAGE_ON_INIT true 系统启动或初始化时自动更新语言项
NONE_SUFFIX_VIEW js,css,map,html... 影响静态/视图资源查找，对多语言页面加载有影响

API 示例（真实路径）

GET /lan/language/list?page=1&limit=20
GET /lan/language/supportedlist
POST /lan/language/updatesupportedlanguage {"name":"en_US","enable":1}
GET /api/getsupportedlanguage
{"code":0,"msg":"success","data":[{"locale":"zh_CN","name":"中文"}]}

页面入口（templates）

  • /modules/lan/language -> templates/modules/lan/language.html
  • /modules/lan/supportedlanguage -> templates/modules/lan/supportedlanguage.html

权限码清单

lan:language:list|info|save|update|delete
lan:language:supportedlist, lan:language:updatesupported

最小可跑通用例

  1. 确认账号具备 lan:language:list 与 lan:language:updatesupported。
  2. 访问 /modules/lan/language，查看语言资源列表。
  3. 调用 GET /lan/language/supportedlist，确认返回支持语言集合。
  4. 调用 POST /lan/language/updatesupportedlanguage 开关一门语言并验证界面切换。
