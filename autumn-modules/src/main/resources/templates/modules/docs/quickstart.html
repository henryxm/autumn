快速开始

目标：让新开发者在最短时间内完成本地启动，并知道下一步看哪里。

1. 环境准备

  • JDK 1.8+
  • MySQL 5.5+
  • Maven 3.0+
  • Redis（可选，分布式部署时启用）

2. 本地启动步骤

  • 创建数据库并配置连接信息
  • 在启动模块执行构建命令
  • 运行启动类并检查启动日志
  • 浏览器访问登录页面完成验证
mvn clean install

3. 验证入口

  • 管理后台：/autumn
  • API 文档：/autumn/swagger/index.html
  • 默认账户：admin/admin（首次部署后请立即修改）

4. 常见问题排查

  • 数据库连接失败：检查用户名密码、字符集、网络权限
  • 端口占用：修改配置端口后重启
  • 页面空白：检查静态资源路径与上下文路径
  • 登录异常：检查验证码、会话与缓存配置
建议在该页后续补充“新同学首日操作清单”，包括创建菜单、配置角色、生成一套模块代码。
