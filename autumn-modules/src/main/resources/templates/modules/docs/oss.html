文件存储（oss）

用于统一文件上传、存储配置与访问策略。

功能入口

  • 对象存储管理：/sys/oss/list、/sys/oss/upload、/sys/oss/delete
  • 云存储配置：/sys/oss/config、/sys/oss/saveConfig

实践建议

  • 按环境隔离 bucket/目录
  • 对外链资源设置过期与权限控制
  • 为大文件上传补充分片策略

关键配置项（来自 sys_config.param_key）

配置键 默认值 说明
CLOUD_STORAGE_CONFIG_KEY {json} 云存储统一配置（阿里云/腾讯云/七牛及类型）
SYSTEM_UPGRADE {json} 系统升级时可能限制上传能力，需联动检查

API 示例（真实路径）

GET /sys/oss/list?page=1&limit=10
GET /sys/oss/config
POST /sys/oss/saveConfig {"type":1,"qiniuDomain":"https://cdn.example.com"}
POST /sys/oss/upload (multipart/form-data)
{"code":0,"msg":"success","url":"https://cdn.example.com/a.png"}

页面入口（templates）

  • /modules/oss/oss -> templates/modules/oss/oss.html

权限码清单

sys:oss:all

最小可跑通用例

  1. 确认账号有 sys:oss:all 权限。
  2. 访问 /modules/oss/oss，查看对象存储列表。
  3. 调用 GET /sys/oss/config 确认当前存储配置可读。
  4. 上传一个测试文件到 /sys/oss/upload，确认返回地址可访问。

故障排查模板

现象 可能原因 处理建议
上传失败 存储凭据无效或大小超限 检查密钥配置与上传限制
文件无法访问 访问域名或权限策略错误 检查对象 ACL 与域名绑定
