用户域能力（usr）

负责用户开放能力、用户令牌与登录行为追踪。

功能入口

  • 用户开放能力：/usr/useropen/*
  • 用户令牌管理：/usr/usertoken/*
  • 登录日志：/usr/userloginlog/*（含 /clean 与 /updateWhite）
  • 用户资料：/usr/userprofile/*

建议补充

  • Token 生命周期与撤销策略
  • 异地登录与高风险行为识别规则
  • 用户隐私字段处理规范

关键配置/参数

配置键 默认值 说明
USER_DEFAULT_DEPART_KEY 空 集群用户同步后的默认部门标识
USER_DEFAULT_ROLE_KEYS 空 集群用户同步后的默认角色（逗号分隔）

API 示例（真实路径）

GET /usr/usertoken/list?page=1&limit=10
GET /usr/userloginlog/list?page=1&limit=20
POST /usr/userloginlog/clean
POST /usr/userloginlog/updateWhite {"id":1,"white":1}
{"code":0,"msg":"success","page":{"list":[...]}}

页面入口（templates）

  • /modules/usr/usertoken -> templates/modules/usr/usertoken.html
  • /modules/usr/useropen -> templates/modules/usr/useropen.html
  • /modules/usr/userloginlog -> templates/modules/usr/userloginlog.html
  • /modules/usr/userprofile -> templates/modules/usr/userprofile.html

权限码清单

usr:usertoken:list|info|save|update|delete
usr:useropen:list|info|save|update|delete
usr:userprofile:list|info|save|update|delete
usr:userloginlog:list|info|save|update|delete

最小可跑通用例

  1. 确认账号具备 usr:userloginlog:list 与 usr:usertoken:list。
  2. 访问 /modules/usr/userloginlog，确认日志列表加载。
  3. 调用 GET /usr/usertoken/list?page=1&limit=10，确认返回 token 列表。
  4. 执行 POST /usr/userloginlog/clean（测试环境）验证清理能力。
